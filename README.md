# 下一个时辰

为你的下一个小时抽一支签。

---

一滴墨珠落在满是文字的纸面上。它有自己的偏好，会主动寻找你心里的那些字——"光"、"远方"、"自由"——然后把它们吞进 Q 弹透明的身体里。被吞掉的字向下坠落，纸面渐渐留下淡淡墨渍。两颗墨珠相遇会融合、溅起水花；在空白处停留太久会像真正的水滴一样蒸发消散。

当碎片积攒到三百个字，一位藏在文字深处的卜者开始工作：从你选的文本残骸中，为你的下一个时辰拾出一句签文。

你在做减法，它在做一道压缩到极致的加法。每一次，都不一样。

## 玩法

1. **选择当下的情绪** — 迷茫、期待、忧虑、平静、躁动、欢喜
2. **写下当下的偏爱** — 随心写几个字，墨珠会主动去寻找它们
3. **选择文本** — 内置自我肯定、唐诗宋词、哲学语录、周易、当代新闻，也可以上传自己的 MD / EPUB / TXT 文件，或者直接粘贴任何文字
4. **叩问此刻** — 点击画布放置墨珠，观察它们漫游、觅食、融合、消散
5. **时辰之签** — 300 字后，AI 从碎片中为你的下一个时辰写一句签文

## 特性

**墨珠精灵**
- Q 弹果冻体：wobble 微颤轮廓 + squash & stretch 运动变形 + 弹簧系统过冲弹跳
- 水滴光学：七层渲染（接触阴影 → 焦散 → 透明体 → 彩虹薄膜边缘 → 双高光 → 纸面反射 → 接触线）
- 有生命感：兴奋时膨胀发光，蒸发时颤抖缩小，身后留一串小水珠轨迹，融合时溅射水花
- 内部可见：吞噬的字在透明身体里缓慢漂流旋转，带鱼眼畸变和明暗梯度

**智能觅食**
- 每种情绪预定义一组偏爱字符 + 用户自定义偏爱字
- 墨珠每 0.35 秒扫描周围 350px，计算加权欲望向量，主动向偏爱字加速
- 偏爱字在画面上以暖棕色微亮标识

**文字物理**
- 四面八方径向引力 + 切向漩涡力
- 被吞噬后上方文字因重力逐列坠落（弹簧阻尼系统）
- 被吞噬处留下墨渍残痕
- 文字深浅不一（alpha 0.25-0.95），墨色从近黑到暖灰，深浅由字符随机分配

**AI 签文**
- 角色：时辰卜者（周易 × 庄子 × 鲁迅 × 纪伯伦）
- 李继刚 Lisp 风格压缩 prompt，追求"读完愣三秒"的签文
- 自动传入当前时辰和卜算时辰
- 兼容 OpenAI 格式 API（默认 Moonshot），无 API 时使用本地种子签文库
- API 设置隐藏（三击顶部装饰线呼出）

**21 项实时调参**
- 右上角 ⚙ 打开调参面板，所有参数拖动即生效
- 物理 7 项：引力、吸引半径、吞噬距离、漩涡、融合距离、消散时间/速度
- 运动 5 项：基础速度、觅食加速、嗅觉范围、回弹力度、坠落重力
- 文字 8 项：字号、字体（13 种可选）、字重、透明度范围、深墨 RGB
- 水滴 5 项：缩小程度、旋转强度、成长速率、初始半径、创作字数

**13 种字体**
- 网络字体：思源宋体、思源黑体、站酷小薇、站酷快乐、马善政楷书、流建毛草、龙藏、芝忙行
- 系统字体：Georgia、宋体、楷体、黑体、仿宋
- 默认龙藏，字重 300，手写感强

**全平台适配**
- 桌面 / iPad / Android 平板 / iPhone / 鸿蒙
- Retina 高清渲染（devicePixelRatio 适配，16MP canvas 上限保护）
- 触摸防穿透、安全区域、横竖屏切换、CDN 双源容灾

## 使用

一个 HTML 文件。双击打开，或拖进浏览器。无需安装，无需服务器。

如需 AI 签文功能，三击设置页顶部装饰线，填入兼容 OpenAI 格式的 API 地址和 Key。不填则使用本地签文库，完全离线可用。
